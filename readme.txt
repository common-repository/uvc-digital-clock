Place a nice LED styled clock on your pages

== Description ==

Simple clocks made in LED style. 4 colors are available. Visit our page at http://ecard.enter-media.org

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Appearance->Widgets to add widget to your pages
